This is the second assignment for Art, Code and Interactivity, manipulating video captures in an interactive way. Utilizing the Kinect to determine people's hands, this project simulates wiping away a window to see through it.

Addons to install for OpenFrameworks:
	- ofxOpenNI https://github.com/gameoverhack/ofxOpenNI (NOTE: OpenNI is a little finicky to get working so I followed the instructions from this video https://www.youtube.com/watch?v=bA3JwQ2KV2M and created a new Addon folder called "ofxOpenNINew")
